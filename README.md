# 💰 Personal Finance Analytics Platform (PFAP)

## Overview

PFAP is an analytics-first personal finance web application designed using enterprise software engineering principles.

The project serves two purposes:

- Daily personal finance management
- Enterprise portfolio project

---

## Technology Stack

### Frontend

- React
- TypeScript
- Vite

### Backend

- FastAPI
- SQLAlchemy
- Alembic

### Database

- PostgreSQL

### Analytics

- PostgreSQL
- Python
- Power BI

---

## Architecture

- Clean Architecture
- Domain-Driven Design (DDD)
- Analytics-First
- Account-Based Ledger
- REST API
